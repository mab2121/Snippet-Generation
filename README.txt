This directory contains the following files:
- Code (All code files)
- Dataset (Manually created dataset of 40 documents with google snippets)
- log_test_runs (Log files with results for different types of test runs)
- Project_Paer (Describes the work done in detail)

To run the code, Please set up the variables exactly as shown in Constants.py
then run the run_all_tests.py module. You may have to install various dependencies
along the way: pip install <dependency>
